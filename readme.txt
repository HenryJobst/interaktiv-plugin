=== interaktiv-plugin ===
Requires at least: 5.3.2
License: MIT License

Das Plugin legt den Inhaltstyp "Interaktiv" an.

== Beschreibung ==

Es wird der Inhaltstyp "Interaktiv" hinzugefügt, welcher bereits von einem Abonnent bearbeitet werden kann.
